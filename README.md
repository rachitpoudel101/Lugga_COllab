
### Installing

Database Configuration
```
- Download the zip file and Unzip the file on your local server.
- Put this file inside “c:/xampp/htdocs/”.
- Open phpmyadmin
- Create a Database named “ecom_store”.
- Import database: /ecom_store.sql
- Open Your browser and put inside the URL: "http://localhost/Lugga_COllab/index.php”
- To Login as admin: ”http://localhost/Lugga_COllab/admin_area”
```
